This is an engine written in C++ that allows for writing using the cursors

This is Windows and Linux compatible (though specific terminal emulators may not work)

# Features
- [ ] Printing
  - [X] Left aligned
  - [ ] Right aligned
  - [ ] Center aligned
  - [X] Print Boundaries
  - [X] Cursor Tracking
- [X] Option Lists
  - [X] Custom highlighting options
- [ ] Grid Maps (WIP)
- [ ] Geometry (WIP)
